# Button - 按钮

## Button 属性

|   属性    |   说明   |  类型   |                 可选值                  | 默认值 |
| :-------: | :------: | :-----: | :-------------------------------------: | :----: |
|   type    |   类型   | String  | 空 、primary、success 、warning、danger |   空   |
| disabled  | 禁用状态 | Boolean |               true、false               | false  |
|   round   | 圆角样式 | Boolean |               true、false               | false  |
| @click="" | 点击事件 |  函数   |                 自定义                  |   空   |
