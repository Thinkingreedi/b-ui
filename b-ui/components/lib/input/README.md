# Input - 输入框

## Input 属性

|   属性   |   说明   |  类型   |   可选值    | 默认值 |
| :------: | :------: | :-----: | :---------: | :----: |
|  value   |  输入值  | String  | 任意字符串  |        |
| disabled | 禁用状态 | Boolean | true、false | false  |
| readonly | 只读状态 | Boolean | true、false | false  |
|  error   | 错误信息 | String  | 任意字符串  |        |
