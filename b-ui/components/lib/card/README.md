# Card - 卡片

## Card 属性

|   属性    |   说明   |  类型   |                 可选值                  | 默认值 |
| :-------: | :------: | :-----: | :-------------------------------------: | :----: |
|   width    |   卡片宽度   | Number  | -- |   0   |
| imgSrc  | 图片地址 | String |               --               | “”  |
|   imgHeight   | 图片高度 | Number |              --              | 0  |
| summary | 图片概述 |  String/slot   |                 自定义                  |   “”   |
